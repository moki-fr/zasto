# Steps to get your OpenRouter API key

## 1. Open link & login

> https://openrouter.ai/

On the first page you'll see "Get API Key" button without even being logged in yet, so click it then login and thats it basically

The steps should bring you to the creation of your API key easily, but if you're not sure , you can find these options manually, follow the next steps

## 2. Get your API key

On the top bar go to "Home", then on the right bar go to API Keys then create your new key.

Just give it a name. The API key should start with `sk-`.
So just copy it and provide it to Zasto by using `zasto.py --key [key]` (or --storekey if you want it to remember your eky)
